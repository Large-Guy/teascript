# Overloading

## Operators

- equals
```js
== (a, b)
```
Overloading of this operator also overloads `!=`

- add
```js
+ (a, b)
```

- subtract / negate
```js
- (a, b)
```

- multiply
```js
* (a, b)
```

- divide
```js
/ (a, b)
```

- mod
```js
% (a, b)
```

- pow
```js
** (a, b)
```

- band
```js
& (a, b)
```

- bor
```js
| (a, b)
```

- bnot
```js
~ (a, b)
```

- bxor
```js
^ (a, b)
```

- lshift
```js
<< (a, b)
```

- rshift
```js
>> (a, b)
```

- and
- or

- greater
```js
> (a, b)
```

- less
```js
< (a, b)
```

- subscript
```js
[] (index, value)
```

## Specials

- tostring
```js
tostring() : string
```

- type
```js
type()
```

- contains
```js
contains(element)
```
